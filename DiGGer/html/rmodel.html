rmodel {DiGGer}R Documentation

Assign a numeric code to a correlation model.

Description

Assign a numeric code to the correlation model: "ID"=0, "AR"=1 or "MA"=2.

Usage

rmodel(rmod)

Arguments

rmod

A DiGGer correlation model code: "ID", "AR" or "MA".
