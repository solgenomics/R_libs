setCorrelation {DiGGer}R Documentation

Modify a Correlation object in a DiGGer search.

Description

Modify a Correlation object in a DiGGer search.

Usage

## S3 method for class 'DiGGer'
setCorrelation(this, phaseNo = 1, objectiveNo = 1, rowModel =
"AR", rowParameter = 0.5, columnModel = "AR", columnParameter = 0.5,
spatialVarianceRatio = 1, ...)
## S3 method for class 'Phase'
setCorrelation(this, objectiveNo = 1, rowModel =
"AR", rowParameter = 0.5, columnModel = "AR", columnParameter = 0.5,
spatialVarianceRatio = 1, ...)
## S3 method for class 'Objective'
setCorrelation(this, rowModel =
"AR", rowParameter = 0.5, columnModel = "AR", columnParameter = 0.5,
spatialVarianceRatio = 1, ...)

Arguments

this

A DiGGer search object.

phaseNo

The Phase number to be modified.

objectiveNo

The Objective number to be modified.

rowModel

"AR" for AutoRegressive, "MA" for Moving Average or "ID" for identity (no correlation).

rowParameter

A value (-1,1) for "AR", (-0.5,0.5) for "MA" or [0.0] for "ID".

columnModel

"AR" for AutoRegressive, "MA" for Moving Average or "ID" for identity (no correlation).

columnParameter

A value (-1,1) for "AR", (-0.5,0.5) for "MA" or [0.0] for "ID".

spatialVarianceRatio

A positive variance ratio.

...

Not used.
