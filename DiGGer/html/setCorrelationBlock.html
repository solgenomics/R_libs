setCorrelationBlock {DiGGer}R Documentation

Set the template for blocks of correlated error.

Description

Set the template for blocks of correlated error in a DiGGer search phase.

Usage

## S3 method for class 'DiGGer'
setCorrelationBlock(this, phaseNo = 1, rowsInCorrelationBlock,
columnsInCorrelationBlock, ...)
## S3 method for class 'Phase'
setCorrelationBlock(this, rowsInCorrelationBlock,
columnsInCorrelationBlock, ...)

Arguments

this

A DiGGer object.

phaseNo

The search phase number to be modified.

rowsInCorrelationBlock

Number of rows in the template correlation block.

columnsInCorrelationBlock

Number of columns in the template correlation block

...

Not used.
