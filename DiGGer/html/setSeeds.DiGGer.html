setSeeds {DiGGer}R Documentation

Set the Random Number Generator Seeds for a DiGGer search.

Description

Set the Random Number Generator (RNG) seeds for a DiGGer Search. The two seeds initialise the RNG.

Usage

## S3 method for class 'DiGGer'
setSeeds(this, s1 = -1, s2 = -1, ...)

Arguments

this

A DiGGer object.

s1

An integer in [0,31328].

s2

An integer in [0,30081].

...

Not used.
